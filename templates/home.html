Credit Card Approval Prediction

This web application takes applicant financial and demographic details, sends them to the trained model, and predicts whether a credit card application is approved or rejected.

The project includes a landing page, a prediction form, and a result page generated by Flask.

Go to Prediction Form

How it works

  • Home page with project overview.
  • Form page collects applicant data.
  • Backend loads the trained model and returns the prediction result.
